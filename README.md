# MapTrace

Add-In para ArcGIS Pro que implementa un sistema de gestión de versiones basado en snapshots para capas vectoriales y raster almacenadas en File Geodatabases.

Desarrollado como trabajo de grado de la Especialización en Sistemas de Información Geográfica — Universidad Distrital Francisco José de Caldas.

## Documentación

La documentación completa está disponible en: **[URL de GitHub Pages]**

## Requisitos

- ArcGIS Pro 3.6 o superior
- .NET 8

## Instalación

1. Descarga el archivo `MapTrace_v1.esriAddinX` desde la sección [Releases](../../releases)
2. Haz doble clic sobre el archivo para instalarlo en ArcGIS Pro
3. Reinicia ArcGIS Pro — la pestaña **MapTrace** aparecerá automáticamente en la cinta
